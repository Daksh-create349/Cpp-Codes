# C++ Lecture Codes

This repository contains a collection of C++ programming examples and exercises organized by lecture. It serves as a resource for learning and practicing various C++ concepts.

## Structure

The codebase is organized into folders for each lecture, containing relevant `.cpp` files.

### Lectures Overview

*   **Lecture 1**: Basic Arithmetic, Data Types, Conditionals
*   **Lecture 2**: loops, patterns
*   **Lecture 3**: Functions
*   **Lecture 4**: Arrays
*   **Lecture 5**: Strings
*   **Lecture 6**: Pointers
*   **Lecture 7**: Recursion
*   **Lecture 8**: Backtracking
*   **Lecture 9**: Object Oriented Programming (OOP)
*   **Lecture 10**: Constructors & Destructors
*   **Lecture 11**: Inheritance
*   **Lecture 12**: Pointers (Advanced) & Memory Management
*   **Lecture 13**: Polymorphism & Exception Handling



## How to Compile and Run

To compile any of the C++ files, you can use a C++ compiler like `g++`.

```bash
cd "Lecture 1"
g++ basic_arithmetic_operations.cpp -o basic_arithmetic_operations
./basic_arithmetic_operations
```
