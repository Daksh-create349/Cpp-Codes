# C++ Lecture Codes

This repository contains a collection of C++ programming examples and exercises organized by lecture. It serves as a resource for learning and practicing various C++ concepts.

## Structure

The codebase is organized into folders for each lecture, containing relevant `.cpp` files.

### Lectures Overview

*   **Lecture 1**: C++ Basics (Arithmetic, Data Types)
*   **Lecture 2**: Conditional Statements (Control Flow, Switch)
*   **Lecture 3**: Loops & Iteration
*   **Lecture 4**: Pattern Printing
*   **Lecture 5**: Number Theory (Primes, Fibonacci, Perfect Numbers)
*   **Lecture 6**: Advanced Number Problems & Jump Statements
*   **Lecture 7**: Introduction to Classes & Objects
*   **Lecture 8**: Object-Oriented Programming (OOP) Practice
*   **Lecture 9**: Functions Introduction
*   **Lecture 10**: Advanced Functions & Practice
*   **Lecture 11**: Utility Functions (Swap, Factorial, Recursion)
*   **Lecture 12**: Pointers & Memory Management
*   **Lecture 13**: Inheritance & Polymorphism

## How to Compile and Run

To compile any of the C++ files, you can use a C++ compiler like `g++`.

```bash
cd "Lecture 1"
g++ basic_arithmetic_operations.cpp -o basic_arithmetic_operations
./basic_arithmetic_operations
```
